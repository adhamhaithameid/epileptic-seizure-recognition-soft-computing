# About This Repository

This repository is the Soft Computing main course project.

## Scope
- Dataset decision and justification
- End-to-end ML pipeline
- Feature reduction and selection techniques
- Multi-model benchmarking
- Result analysis and research paper drafting

## Course Methods Included
- Reduction: PCA, LDA, SVD
- Selection: Filter (ANOVA), Wrapper (SFS), Embedded (L1), Genetic Algorithm
- Models: KNN, SVM, Decision Tree, Logistic Regression, LDA Classifier, MLP ANN
- Metrics: Accuracy, Precision, Recall, F1, ROC-AUC, confusion matrices

## Execution Modes
- Local Python workflow (`run_all.sh`)
- Google Colab notebook (`03_notebooks/colab/epileptic_seizure_full_pipeline_colab.ipynb`)
