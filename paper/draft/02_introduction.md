# Introduction

Epileptic seizure recognition is an important biomedical classification task where reliable automation can assist earlier clinical decisions. Existing studies report strong performance, but comparison across papers is often difficult because preprocessing, feature engineering, and validation protocols differ.

This work addresses that gap by building a reproducible, course-aligned Cartesian benchmark framework that evaluates all requested soft-computing stages under one protocol: preprocessing, reduction, selection, multi-model benchmarking, and paper-ready reporting. The framework is designed to make method comparisons transparent, repeatable, and easy to audit.
